# Frontend Blade Integration (UI-Only)

Folder ini melanjutkan phase frontend menjadi struktur Blade-ready.

## Struktur
- `resources/views/layouts` untuk layout publik dan admin
- `resources/views/components` untuk komponen reusable
- `resources/views/public` untuk halaman visitor
- `resources/views/admin` untuk halaman admin CMS (UI-only)

## Catatan
- Fokus tetap frontend saja, belum ada controller/model/database.
- Asset sementara memakai file yang sudah ada di `frontend/assets`.
