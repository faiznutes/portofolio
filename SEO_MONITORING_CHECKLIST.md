# SEO Monitoring Checklist

## Setelah Deploy (Hari 0)

- Submit ulang `https://faiznute.site/sitemap.xml` ke Google Search Console.
- Gunakan URL Inspection untuk request indexing halaman: `/`, `/works`, `/services`, `/cv`, `/contact`.
- Validasi robots live: `https://faiznute.site/robots.txt`.
- Validasi canonical live per halaman publik.
- Cek redirect `.html` ke clean path bekerja (301).

## Monitoring 7 Hari

- Pantau query branded: `faiznute`, `portofolio faiznute`.
- Pantau query service intent: `web dev`, `editor`, `video editor`, `jasa video editor`, `portofolio web dev`.
- Cek CTR halaman utama (`/`) dan halaman layanan (`/services`).
- Cek coverage issue baru di Search Console.

## Monitoring 14 Hari

- Bandingkan impresi dan klik branded vs non-branded.
- Cek halaman yang mulai ranking untuk long-tail query.
- Review dan update title/description jika CTR rendah.
- Putuskan prioritas lanjutan untuk Phase 2 optimization berikutnya.
